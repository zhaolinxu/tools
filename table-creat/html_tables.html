如何使用它？

  1. 使用“ 表” 菜单设置表 的所需大小。
  2. 在表格中输入表格数据：
    • 从电子表格中选择并复制（Ctrl + C）表格（例如Google Docs，LibreOffice Calc，网页）并将其粘贴到我们的编辑器中 - 单击单元格，然后按Ctrl + V
    • 或者只需 双击任何单元格 即可开始编辑它的内容 - Tab键和箭头键可用于浏览表格单元格
  3. 使用菜单中的选项和使用工具栏按钮调整文本对齐和表格边框 - 格式化应用于所有选定的单元格。
  4. 单击“生成”按钮查看生成的表的HTML源代码 - 选择它，然后复制和粘贴到您的网站的来源。

我们希望这个工具对于不太熟悉HTML和CSS的人来说将是有用的。 所以如果你需要一个桌子为您的网站或博客（Wordpress，Drupal或任何平台，允许放置HTML代码在帖子），它应该工作正常。 我们的HTML表生成器对于只需要快速创建HTML表的开发人员也是有用的。 请注意，生成的表的代码中保留换行符。

备注

要将表格插入您的网站，只需将生成的代码复制并粘贴到您的网站的源代码中即可。 它应该在桌面和移动的所有现代浏览器中显示良好。 但是，如果您希望您的页面与HTML标准保持一致，请阅读下一段。

生成的代码由 <style> 标签和 <table> 标签 两部分组成 。 第一个应该被复制并放在 </head> 网站 的 标签 之前 ，而后者（即表格代码）应放在所需的位置。

表主题

您可能注意到，工具栏中有一个选择框“ - 表主题”，它允许从预定义集中选择表格主题。 生成的CSS包含所有必需的颜色等，以便在将其粘贴到您的网站时，表格应该类似。

Quick help

Basic key commands

  • ENTER or double click — to start editing a cell
  • ESC — to stop editing a cell
  • TAB or Arrow Keys — to select a different cell
  • CTRL+Z — to undo a change
  • CTRL+Y — to redo a change
  • Left mouse button — click a cell to select it (hold left button pressed to select adjacent cells)
  • Hold CTRL to select multiple cells (not necessarily adjacent).

Some of the functionalities may not work in all browsers — if you encounter a problem, please try to use another browser (Google Chrome, Mozilla Firefox) or update your browser to a newer version if available. If the problem still persists please let us know.
